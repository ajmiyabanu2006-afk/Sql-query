# Sql-querySQL Query Generator
Project Overview
This project is developed as part of the Naan Mudhalvan program.
The SQL Query Generator is designed to convert natural language input into SQL queries automatically using Artificial Intelligence.
Normally, writing SQL queries requires knowledge of database languages. This project makes it easier by allowing users to type their request in simple English, and the system generates the corresponding SQL query.
Objectives
To simplify SQL query creation.
To help beginners interact with databases easily.
To convert natural language questions into SQL queries automatically.
Technologies Used
Python
Natural Language Processing (NLP)
Google Colab
IBM Granite 3.3 2B
How the Project Works
The user enters a question in natural language.
The system processes the input using an AI model.
The model generates the corresponding SQL query.
The output SQL query is displayed to the user.
Features
Converts natural language to SQL queries
Easy to use for beginners
Reduces the need for manual SQL coding
Helps in learning SQL queries easily
Future Scope
Integration with real databases
Web-based user interface
Support for more complex queries
Conclusion
The SQL Query Generator project helps users generate SQL queries easily from natural language input. This system improves accessibility to database operations and supports beginners in learning SQL more efficiently.
